Step 1 — API Token

Not connected

Enter your product's API token, or click Generate Token to create a demo token. Then click Connect to verify.

📖

Step 2 — Configure OpenClaw

  1. Open your OpenClaw config file:
    ~/.openclaw/config.json
  2. Add the custom channel block with your token: 
    {
  "channels": {
    "custom": {
      "enabled": true,
      "transport": "webhook",
      "webhook": {
        "port": 3100,
        "path": "/webhook/custom"
      },
      "dmPolicy": "open",
      "accounts": {
        "main": {
          "apiBaseUrl": "https://api.your-product.com",
          "apiToken": "YOUR_TOKEN",
          "webhookSecret": "optional-secret"
        }
      }
    }
  }
}
  3. Restart the OpenClaw gateway:
    openclaw gateway --restart
  1. Install the channel plugin:
    npm install openclaw-channel-custom
  2. Set the API token:
    openclaw config set channels.custom.accounts.main.apiToken "YOUR_TOKEN"
  3. Enable the channel:
    openclaw config set channels.custom.enabled true
openclaw config set channels.custom.transport webhook
  4. Restart:
    openclaw gateway --restart
